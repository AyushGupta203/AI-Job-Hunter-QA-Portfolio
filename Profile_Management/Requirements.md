# Profile Management Module Requirements

## Feature

Profile Management

## Fields

- LinkedIn URL
- GitHub URL
- Portfolio URL
- LeetCode URL

## Validations

- User can view profile details
- User can update profile information
- Validation of LinkedIn URL format
- Validation of GitHub URL format
- Validation of Portfolio URL format
- Validation of LeetCode URL format
- Whitespace-only values should not be accepted for external links
